    Remark:
    To execute the repository miner on OSX, the "und" command line tool must first be added to path.
    1. Create a .zshrc file in your user folder (for OSX Catalina zsh is the default shell)
    2. add path to understand cmd tool to zsh configuration
       export PATH=/Applications/Understand.app/Contents/MacOS:$PATH
       See here for guide:
       https://coolestguidesontheplanet.com/how-to-add-to-the-shell-path-in-macos-using-terminal/
